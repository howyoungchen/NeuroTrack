# NeuroTrack / 心迹

[English](README.md)

一个安静的 Android 小应用，帮你在焦虑症等神经症的康复期，留意自己的压力和睡眠有没有悄悄变差。

所有数据都存在你自己的手机里，不上传、不注册、不打扰。它不是医疗软件，不会给你下诊断——它只是帮你"看见"自己的状态。

## 为什么会有这个项目

如果你经历过焦虑症或类似的神经症，你大概知道：康复不是终点，复发的风险会一直悬在那里。

而复发往往不是突然发生的。在真正"垮掉"之前，通常早就有征兆了——连续几晚睡不好、莫名的紧绷感、又开始反复琢磨一些事。问题是，身处其中的人很难意识到这些信号，等发现的时候，焦虑已经扩散开了。

我想要一个东西，能在这个阶段拉自己一把：把这些早期信号记录下来、算出趋势，在压力真的在爬升时提醒我一声。这样就有机会在焦虑扩散之前主动踩刹车，而不是事后才反应过来。

市面上没找到合适的，所以我自己写了一个。

顺便说一句设计上的坚持：**这个 App 本身绝不能成为新的压力来源。** 所以它很克制——不会天天弹通知催你打卡，只在真正需要的时候出声。

## 它能做什么

- **每周自测**：10 道简单的选择题，一两分钟就能做完
- **压力评分**：结合自测结果和睡眠信号，算出 0-10 分的压力等级
- **压力预警**：只有分数超过 5 才会通知你，平时保持安静
- **睡眠观察**：通过屏幕亮灭的时间戳推断你的作息（只记录时间点，绝不读取屏幕内容），展示昨天、最近一周、最近一个月的睡眠情况
- **温和的提醒**：如果你很久没自测了，每周提醒你一次
- 支持中英文、深色/浅色主题

## 怎么用

1. 打开 App，在 **状态** 页看到当前的整体情况
2. 去 **自测** 页做一次自查，让 App 知道你最近的主观感受
3. （可选）在 **设置** 里开启睡眠监测，App 会开始通过屏幕亮灭推断你的作息
4. 之后就顺其自然——偶尔回来看看趋势，压力升高时 App 会主动告诉你

就这么简单。它不需要你每天投入精力去"经营"，用得越省心越好。

## 隐私

这类数据很敏感，所以：

- 所有记录（自测、睡眠、屏幕事件、日志）只存在本机
- 不联网、不上传、没有账号系统
- 睡眠监测只记录屏幕点亮/熄灭的时间戳，不碰屏幕内容、消息、位置或任何其他应用的数据
- 日志只有你手动点导出才会离开手机

需要的权限也只有几个：通知（发提醒和预警）、开机自启（重启后恢复定时任务）、忽略电池优化和精确闹钟（让监测和提醒更可靠，都是可选的）。

## 一个重要的提醒

NeuroTrack 只是一个自我观察的工具，**它不能替代医生**。

如果你正在经历严重的症状、持续恶化，或者有伤害自己的想法，请立刻联系专业人士或当地的紧急救助服务。照顾好自己，永远是第一位的。

## 从源码构建

需要 Android Studio 2025.3+、JDK 17+（AGP 9.2.1，Min SDK 26 / Target SDK 36）：

```powershell
.\gradlew.bat assembleDebug
```

提 PR 前建议跑一下：

```powershell
.\gradlew.bat :app:compileDebugKotlin
.\gradlew.bat :app:lintDebug
.\gradlew.bat :app:testDebugUnitTest
```
