# NeuroTrack

NeuroTrack 是一个本地优先的 Android 周压力观察与正念练习工具。

它不再分析睡眠、读取应用使用情况或位置，也不按天给压力下结论。应用把一次过去七天的完整回顾与本周正念完成情况结合，形成一个周压力分数，并只在状态页呈现当前压力条和最近八周趋势。

## 核心体验

- **每周回顾**：10 个具体问题覆盖睡眠感受、手机注意力、观点弹性、重大事件、环境与自我整理、工作学习、人际联系、身体信号、恢复时间和未完成事项带来的担忧。
- **周压力**：周回顾给出基础分；周一、周三、周五、周日计划的正念缺席会作为恢复不足，最多增加 2 分。
- **正念练习**：可选 5、10 或 15 分钟，开始后播放本地生成的舒缓音乐、保持亮屏并屏蔽返回键。离开应用或切换到其他应用会将本次练习标记为中断。
- **固定节奏**：每周一、三、五、日练习，提醒时间可在设置中调整。
- **最少权限**：仅在 Android 13 及以上请求通知权限，用于正念提醒。
- **完全本地**：没有账号、网络上传、分析 SDK 或云备份；数据库升级直接重建，不保留旧版睡眠数据兼容。

## 页面

- **状态**：本周压力条、最近八周压力趋势。
- **练习**：本周节奏、过去一周回顾、正念练习。
- **设置**：提醒时间、通知权限、语言和主题。

## 构建与验证

项目需要 JDK 17+、Android SDK 36，从仓库根目录运行：

```powershell
.\gradlew.bat :app:testDebugUnitTest
.\gradlew.bat :app:compileDebugKotlin
.\gradlew.bat :app:lintDebug
.\gradlew.bat assembleDebug
```
